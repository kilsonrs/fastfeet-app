## Para baixar e executar o projeto, siga esses passos:
<br />
<br />

### Baixe o repositório
```bash
git clone https://github.com/kilsonrs/fastfeet-app.git
```
<br />

### Acesse o diretório
```bash
cd fastfeet-app/
```
<br />

### Baixe as dependências do projeto
```bash
yarn install
```
<br />

### Por fim, execute o comando
```bash
yarn android
```
